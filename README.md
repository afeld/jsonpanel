## Development

### Setup

```bash
gem install bundler
bundle
npm install -g bower
bower install
```

### Compile files

```bash
bundle exec compass watch
open index.html
```

### Run test coverage

```bash
bundle exec rackup
open "http://localhost:9292/index.html?coverage=true"
```
